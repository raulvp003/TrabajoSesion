# Proyecto SignUp SignIn in Application

## Información General
Este proyecto consiste en el desarrollo del **lado cliente (Front-End)** de una aplicación web bancaria que permite realizar los siguientes casos de uso:

- **Registro de usuario (Sign Up)**
- **Inicio de sesión (Sign In)**
- **Cambio de contraseña (Change Password)**

El objetivo es emular la interfaz gráfica de un sistema de autenticación bancaria, comunicándose con un **back-end RESTful** ya proporcionado.  
Este trabajo forma parte de las asignaturas **Diseño de Interfaces Web** y **Desarrollo Web en Entorno Cliente** del ciclo formativo de **Desarrollo de Aplicaciones Web**.

El codigo fuente se encuentra en el siguiente repositorio:
- https://github.com/raulvp003/TrabajoSesion

## Estructura 
Este repositorio Git tiene la siguiente estructura:
- Carpeta ‘web’ que contiene todo el código de la aplicación web
- Dentro de la carpeta web tenemos:
    - ‘index.html’ junto a su respectivo CSS ‘CSSindex.css’
    - Usamos ‘CSSlogin.css’ como estilo base para cada CSS de cada panttalla
    - Carpeta ‘signup’ que contiene el codigo necesario para el registro del usuario
- Carpeta ‘signin’ que contiene el codigo necesario para el inicio de sesion y la carpeta ‘Change Password’ que contiene ‘CambioContraseña.html’ ,‘Csslogin.css’ y ‘Ccjs.js’

Nuestras principales tecnologías utilizadas para este proyecto han sido HTML5,CSS3 y JavaScript

## Diseño
Para este diseño hemos decidido hacer una interfaz atractiva pero a la vez sobria para adecuarnos a lo que espera un consumidor de una app de banca y hemos ajustado a este propósito los colores y tipografías. Nos hemos inspirado en varias plantillas de diseño adoptando de ellas lo que veíamos interesante y hemos depurado y testeado el codigo en los principales navegadores de pc y movil.
